Starting with Linux Lite

Install Updates

Before you can begin using Linux Lite, your first need to run Install Updates.

Click on Menu, Favorites, Install Updates. On the next window that pops up, enter the password you created for your user during the installation. Don't worry if you can't see your password as you type it, this is a security feature and prevents prying eyes from seeing what you type.

Release Notes

In this release we wanted to combine the newest versions of well established and supported software like LibreOffice, VLC, Wine and Gimp so that people have access to the latest features in those programs. In Linux Lite 2.2 you get the carefully crafted combination of current software in certain programs on top of a stable operating system. We think this is the happy medium people are looking for between long term support whilst also having 'the newest stuff'. Thank you to everyone who contributed ideas, code and their support.

Linux Lite 2.2 is released with the following features:

  • Firefox 33, Thunderbird 31.2.0
  • Kernel 3.13.0-24
  • Whisker Menu
  • Newest versions of: LibreOffice, VLC, Wine, Gimp and many more
  • New Backup software - Menu > Accessories > Backups
  • New Date & Time software - Menu > Settings > Date & Time
  • New Lite Cleaner application - clean your system. Find it under Menu > System > Lite Cleaner
  • New and improved Install Additional Software and Remove Additional Software
  • New 'Check Install Media' added to the live boot image
  • New Linux Lite themed wallpapers
  • Support link directly from, Menu > Internet > Support

Hardware Recommendations

Linux Lite can run on a wide range of hardware. Our online Hardware Database contains a growing list of computers that can run Linux Lite. The lists below contains 2 recommendations. The first - a set of minimum recommended specs, the second - preferred specs.

Minimum Recommended Specs

  • 700MHz processor
  • 512mb ram
  • VGA screen capable of 1024x768 resolution
  • DVD drive or USB port for the ISO image

Preferred Specs

  • 1.5GHz processor+
  • 1024mb ram+
  • VGA, DVI or HDMI screen capable of 1366x768 resolution+
  • DVD drive or USB port for the ISO image
